 CityLife 360 — Power BI Dashboard Project

A comprehensive data analytics dashboard exploring urban life patterns, city metrics, and key performance indicators across multiple dimensions.


📊 Project Overview
CityLife 360 is an interactive Power BI dashboard designed to provide a 360-degree view of city life data. The project focuses on transforming raw urban datasets into meaningful visual insights, supporting data-driven decision-making for city-related analysis.

🛠️ Tools & Technologies
Tool Purpose
Power BI Desktop - Dashboard development & visualization
Microsoft Excel - Data preparation & cleaning
DAX - Calculated measures & KPIs
Power Query - Data transformation & ETL

✨ Key Features

📌 Interactive KPI Cards — High-level metrics at a glance
📈 Trend Analysis — Time-based visual breakdowns
🔍 Dynamic Filters & Slicers — User-driven data exploration
📊 Multi-page Dashboard Layout — Organized reporting structure
🎨 Clean UI Design — Professional and presentation-ready visuals

Download and install Power BI Desktop (free)
